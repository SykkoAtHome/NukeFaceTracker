MediaPipe FaceMesh Landmark Grouper

Parent -> child hierarchy. Landmarks are assigned only to child groups.

Profile

Each profile has its own parent and child groups. Export stores all profiles together.

Parent group

A parent is only a container, such as nose. Assign landmarks to child groups.

Child group

Clicked landmarks go to the active child group. The same landmark can be shared by multiple child groups.

Groups

Hover a point
Scroll = zoom. Drag background = pan; in Free 3D, drag = rotate and Shift + drag = pan. Click = assign to active child group. Alt + click = remove from active child group.

Export format
